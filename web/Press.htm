  • "Agender - free, fast and easy to use note reminder tool for Windows users."
    Agender is a light, free utility that allows you to organize all your thoughts comprehensively. Using the application is quite straightforward: simply select one of the dates in the calendar, click Add Note and type in your note title. Agender loads and saves automatically so you don't have to. Select the opacity level and customize the notes color. Built in C++, Agender runs very smoothly and is available in english, spanish, deutsch and japanese.
    Main Features of Agender:
    - GPL version 3+ licensed
    - Loads and saves automatically
    - Tray icon for fast access
    - Y2K compliant
    - Manages dates from 4,300 B.C. to 10,000 A.D.
    - Sticky notes(showed all time)
    - Multilingual, already available in english, spanish, deutsch and japanese
