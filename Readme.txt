Dependencies:
	To compile you will need wxWidgets 2.8, I'm using 2.8.10 maybe 
	it works with another. The website is www.wxwidgets.org.

Compiling:
	You can use the Code::Blocks project included in the package or 
	use the Makefile, type make.
Installing:
	type su -c 'make install'
